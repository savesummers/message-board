# 📝 Message Board Project

A simple Message Board web application built with **Django** that displays messages stored in a database. This project demonstrates the use of Django's generic class-based views, models, templates, and URL routing.

## 🚀 Features

- Display messages from a database
- Django Generic `ListView`
- SQLite database
- Django Template Language (DTL)
- Django Admin support
- Beginner-friendly project structure

## 🛠️ Technologies Used

- Python 3
- Django
- SQLite3
- HTML5
